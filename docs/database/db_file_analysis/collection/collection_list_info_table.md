# collection_list_info_table

收藏信息。

| 列名 | 类型 | 含义 | 注 |
| --- | --- | --- | --- |
| 180001 | str | sid | 用于确定唯一性 |
| 180008 | int | type | 详见[收藏类型](#收藏类型) |
| 180011 | int | last updated time | 最后编辑时间 |
| 180004 | protobuf | Source Info | 详见[来源信息](#来源信息) |
| 180009 | int | first time | 创建/收藏时间 |
| 180015 | protobuf | summary info | 详见[摘要信息](#摘要信息) |

### 收藏类型

| 值 | 注 |
| --- | --- |
| 1 | 来源于聊天记录（文本类消息） |
| 2 | QQ 笔记 |
| 9 | 带有链接的聊天消息 |

### 来源信息

| Field Number | 注 |
| --- | --- |
| 18504 | 收藏来源群号 |
| 18505 | 群名称 |
| 18506 | 发送者 nt_uid |
| 18501 | 发送者 QQ 号 |
| 180503 | 备注名/群员名称/用户昵称 |

### 摘要信息

| Field Number | 注 |
| --- | --- |
| 181450 | 标题（可为空） |
| 181452 | 外显描述内容 |
| 181453 / 18504 / 180550 / 180553 | 图片信息 / 来源会话 / 图片 URL / 图片名称 |
| 180561 | 图片本地缓存路径 |
| 180610 | 文件保存位置 |
