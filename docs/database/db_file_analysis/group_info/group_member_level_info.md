# group_member_level_info

群等级头衔信息。

| 列名 | 类型 | 含义 | 说明 |
| --- | --- | --- | --- |
| 60001 | int | 群号 | |
| 67100 | int | 群等级 | |
| 67103 | protobuf | 群头衔映射表 | protobuf，等级序号到头衔名称的映射，详见[下表](#_67103-值解读-群头衔映射) |

## 67103 值解读（群头衔映射）

`67103` 为一段 protobuf，外层为可重复字段 `67103`，每个等级对应一个「序号→头衔名」子结构（如 1→潜水、2→冒泡）：

| Field Number | 类型 | 含义 | 说明 |
| --- | --- | --- | --- |
| 67200 | int | 等级序号 | 如 1、2、3… |
| 67201 | string | 头衔名称 | 该等级对应的头衔文字 |
