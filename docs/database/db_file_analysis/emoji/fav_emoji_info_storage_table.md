# fav_emoji_info_storage_table

QQ 收藏表情数据。

| 列名 | 类型 | 含义 | 说明 |
| --- | --- | --- | --- |
| 80002 | str | 文件名 | 格式为 `{uin}_0_0_0_{MD5}_0_0` |
| 80001 | int | 表情排序 | 倒序，即最新的收藏表情序号为 0 |
| 1002 | str | 收藏用户 uin | |
| 80012 | str | 本地存贮路径 | |
| 80010 | str | 下载地址 | 格式为 `https://p.qpic.cn/qq_expression/{1002}/{80002}/0` |
| 80011 | str | 表情 MD5 | 32 位大写 |
| 80213 | int | 是否为原创表情 | 0 为否，1 为是 |
| 80201 | str | 原创表情标识符 | 当 80213 为 1 时 |
| 80202 | int | 原创表情包 ID | 当 80213 为 1 时 |
| 80223 | str | 表情备注 | |
| 80225 | str | 表情备注 | |
