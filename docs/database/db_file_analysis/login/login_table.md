# login_table

已登录账号列表，一行为一个账号。

| 列名 | 类型 | 含义 | 说明 |
| --- | --- | --- | --- |
| 1000 | int | uin | QQ 号 |
| 1001 | str | uid | nt_uid（长 UID，路由用） |
| 1002 | str | 头像链接 | 缓存的头像 CDN 地址 |
| 1007 | str | 昵称 | 账号显示名 |
| 1004 | bytes | A1 登录票据 | A1 key（登录凭证），以字节存贮，通常转 hex |
| 1008 | int | 最近登录时间 | 秒级时间戳，从未登录为 0 |

> [!WARNING]
> `1004`（A1 登录票据）为敏感登录凭证，泄露可能导致账号被登录，请勿分享包含此列的数据。
