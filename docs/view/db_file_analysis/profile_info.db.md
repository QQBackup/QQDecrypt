#### 20072值解读（密友关系）

`20072` 为一段 protobuf，记录本账号与该好友设置的「密友关系」（情侣 / 闺蜜 / 死党等）。外层字段 `20072` 内含以下子字段：

| Field Number | 类型      | 含义         | 说明                                             |
| ------------ | --------- | ------------ | ------------------------------------------------ |
| 20073        | int（可重复） | 预设关系ID   | 已设置/预选的关系ID列表                          |
| 20074        | int       | 当前展示关系ID | 主页当前展示的关系；未单独设置时取 20073 的首项 |

关系ID与名称的对应关系如下（未知ID一般为「游戏搭子」一类的普通搭子关系）：

| 关系ID | 名称     |
| ------ | -------- |
| 1      | 情侣     |
| 2      | 闺蜜     |
| 3      | 基友     |
| 26     | 死党     |
| 81     | 学习搭子 |
| 82     | 运动搭子 |
| 101    | 饭搭子   |

> [!NOTE]
> 关系ID编号会随 QQ 版本增补，上表为已知的常见值，不保证完整。
